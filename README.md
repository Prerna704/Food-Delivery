lazeez – Food Ordering Website

This repository hosts the source code for LAZEEZ, a dynamic food ordering website built with the MERN Stack. It provides a smooth and user-friendly platform for online food ordering.



Features

User Panel:

Login / Signup

JWT Authentication

Password Hashing with Bcrypt

Add to Cart

Place Order

Stripe Payment Integration

Logout

Admin Panel:

Products Management

Order Management

Role-Based Identification

Shared Features:

Authenticated APIs

REST APIs

Filter Food Products

Beautiful Alerts
